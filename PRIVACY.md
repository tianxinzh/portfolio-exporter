# Privacy Policy — Portfolio Exporter

Last updated: 2026-06-01

Portfolio Exporter runs entirely in your browser.

- **What it accesses:** while you are on robinhood.com, it reads the holdings data
  your browser already receives from Robinhood (equities, options, crypto, and
  their prices) in order to convert it into a file you download.
- **What it stores:** only your export preferences (chosen format, columns,
  filters, account label) via Chrome's `storage` API. Nothing else is retained.
- **What it sends:** nothing. There are no servers, no analytics, no trackers, and
  no remote code. Your holdings never leave your computer except as the file you
  choose to download.
- **Permissions:**
  - `storage` — remember your settings.
  - `downloads` — save the export file you request.
  - access to `robinhood.com` — read the holdings on the page you're viewing.

Not affiliated with or endorsed by Robinhood. Contact: open an issue on the
project's GitHub repository.
